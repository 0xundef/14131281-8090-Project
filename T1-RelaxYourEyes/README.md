# Task 1: Relax Your Eyes

## Overview
This application monitors eye fatigue by tracking user activity (keyboard and mouse input) and enforces break times by locking the screen.

## Features
- **Input Monitoring**: Tracks keyboard and mouse usage to detect activity.
- **Screen Locking**: Automatically locks the screen to force a break.
- **Configuration**: Customizable settings via `config.yaml`.

## Usage
Run the application from the project root:

```bash
python3 T1-RelaxYourEyes/main.py
```
